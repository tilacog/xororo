XPlit

One-Time Pad (OTP) 2-of-2 Secret Sharing

DEMO ONLY
Do not use with real sensitive data. This tool runs entirely in your browser, but you should never enter real secrets into any web application. Use the CLI tool for production use.
How it works: XPlit implements a one-time pad (OTP) using XOR with cryptographically random data. Both shares are required to recover the secret. One share alone reveals zero information about the secret (information-theoretic security). CRC32 checksums detect corrupted shares.
